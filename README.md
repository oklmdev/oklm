## Introduction

Boilerplate pour un produit web pour les pros du web

Architecture:
- Clean / Hexagonal Architecture
- CQRS / Event Sourcing
- MPA (SSR + génération d'un bundle par page interactive)
- Monorepo simple (pas besoin de lerna/yarn workspaces en MPA)

Environnement:
- TypeScript
- Jest
